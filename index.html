Apis

Based on classes and Operating System

An API, for its acronym Application Programming Interface, is nothing more than a set of rules that describe how one application can interact with another, as well as the mechanisms so that this communication can be carried out.

Class based API

What is class-based API??

These allow union with data around classes, usually object-oriented with JAVA. It uses abstract classes to create apps just like any program developed in this language.

These types of APIS allow us to make use of already defined classes which already have programming logic, so that we can speed up our software creation process.

Example

An Example is the Java API, which is nothing more than a set of Classes, organized in packages, which already have programming logic, Attributes, methods, etc...

In this API we can find classes such as System, Math, String etc...

Operating system APIs.

What are operating system APIs?

This group of APIs define how applications use the available resources and services of the operating system. So each OS (Operative System) has a set of APIs, for example, Windows API or Linux API have the kernel-user space API and kernel internal API.

Example

This type of APIS allows our programs to interact with the Operating System, in such a way that we can have access to elements such as the keyboard, mouse, graphical interface, we can create processes, threads, among others

Conclusion

APIs are an excellent way in which we can communicate between two pieces of software, which will allow us to standardize our applications and of course not rewrite code.
